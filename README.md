# TaroomTool

Interactive 3D schematic of the Taroom Trough (Surat Basin overlying Bowen Basin), built with Three.js. Shows stratigraphy, groundwater systems, hydrocarbon targets, and well infrastructure.

## View / Share

Live link (once GitHub Pages is enabled): **https://gschoning.github.io/TaroomTool/**

To enable hosting:
1. Go to **Settings → Pages** in this repository
2. Under **Build and deployment → Source**, select **Deploy from a branch**
3. Choose branch `main` (or whichever branch you merge to) and folder `/ (root)`
4. Save — the URL above will be live within a minute

The whole tool is a single static `index.html`, so anyone with the link can open it in a browser; no server needed.
